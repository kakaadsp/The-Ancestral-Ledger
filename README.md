# 🌿 The Ancestral Ledger

### *Kedaulatan Biologis. Komersialisasi Digital.*

## 🎯 Latar Belakang

Kekayaan biologis dan pengetahuan tradisional Nusantara saat ini berada dalam ancaman asimetris. Praktik **biopiracy** telah berevolusi dari pencurian fisik menjadi ekstraksi **Digital Sequence Information (DSI)** — data genomik mikrobiota lokal (seperti kapang tempe *Rhizopus oligosporus*, dan dadiah *Lactobacillus plantarum*) seringkali dimanfaatkan oleh korporasi multinasional **tanpa mekanisme Benefit-Sharing yang adil**.

**The Ancestral Ledger** hadir sebagai *governance layer* berbasis Blockchain yang mengamankan data mentah sekuens genetik dari fasilitas riset (BRIN) menjadi aset digital B2B yang **kebal manipulasi**, sekaligus mewajibkan korporasi global membayar royalti yang distribusinya dieksekusi secara instan dan transparan kepada masyarakat adat dan pengrajin lokal.

## 🏗️ Arsitektur Ekosistem

```
┌─────────────────────────────────────────────────────────────────────┐
│                    THE ANCESTRAL LEDGER                              │
│                                                                     │
│  [01] Biodiversitas  →  [02] Sekuensing  →  [03] Blockchain        │
│  Sampel pengrajin        NGS BRIN             Immutable Prior Art   │
│       ↓                                              ↓              │
│  [06] Distribusi  ←  [05] Global B2B  ←  [04] Smart Contract       │
│  Royalti Real-time      Lisensi B2B          Digital MTA & ABS     │
└─────────────────────────────────────────────────────────────────────┘
```

**Alur Nilai DSI:**
1. **Pengrajin Lokal** menyediakan sampel mikroba dan pengetahuan tradisional
2. **BRIN** melakukan ekstraksi *Next-Generation Sequencing* (NGS)
3. **Blockchain** mengamankan data sebagai *Prior Art* immutable (WIPO ST.26)
4. **Smart Contract** menjalankan Digital MTA & distribusi ABS otomatis
5. **Korporasi Global** mengakses lisensi legal untuk R&D biofarmasi/F&B
6. **Pengrajin Lokal** menerima royalti 85% secara real-time

## 🚀 Quick Start

### Prerequisites
- Node.js ≥ 18.x
- npm ≥ 9.x

### Installation

```bash
# Clone repository
git clone https://github.com/kakaadsp/The-Ancestral-Ledger.git
cd The-Ancestral-Ledger

# Install dependencies
npm install

# Start development server
npm run dev
```

Buka browser di **http://localhost:5173** 🎉

### Build for Production

```bash
npm run build
```

Output tersedia di folder `dist/`.
